# Prédiction du prix de vente de biens immobiliers

## Objectif
Ce projet de data science a pour objectif de prédire le prix de vente de maisons à partir de leurs caractéristiques afin d’aider une agence immobilière à estimer plus finement la valeur des biens et à optimiser sa stratégie de tarification.

## Contexte métier
RealAgents est une agence immobilière qui souhaite améliorer la fixation du prix de mise en vente de ses biens. Une estimation plus précise du prix de vente permet de réduire le temps passé sur le marché et de limiter les ajustements de prix tardifs.

## Problématique
Comment prédire le prix de vente d’un bien immobilier à partir de variables comme la ville, la surface, le nombre de chambres, le type de maison ou encore la durée de mise en vente ?

## Structure du projet
- `data/raw/` : données brutes
- `data/processed/` : données nettoyées
- `notebooks/` : notebooks d’analyse et de modélisation
- `outputs/` : résultats, prédictions et graphiques
- `src/` : fonctions Python réutilisables
